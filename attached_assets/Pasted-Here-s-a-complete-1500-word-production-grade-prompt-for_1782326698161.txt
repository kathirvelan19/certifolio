# ЁЯЪА CERTFOLIO тАУ FULL-STACK WEB APPLICATION MASTER PROMPT

## 1. Project Overview

Build a full-stack web application called **тАЬCertfolioтАЭ** that allows students to upload academic, internship, and certification documents (PDF or images), generate secure shareable public links, and showcase them in a professional portfolio format.

The main goal is to solve a real-world problem: students struggle to prove their skills and certifications during placements and job applications. Certfolio provides a centralized, verifiable, and shareable certificate management system.

Recruiters should be able to open a link and instantly verify certificates without logging in.

---

## 2. Core Idea

* Students upload certificates (PDF or images)
* System stores files securely in Cloud storage
* Generates a public URL for each certificate
* Users can share this URL in resumes, LinkedIn, GitHub, or portfolios
* Recruiters can view certificates without login
* Each certificate tracks views and metadata
* Users also get a personal portfolio page
